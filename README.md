# Airline Bundle Calculator

Demo de calculadora de bundles para aerolíneas.

## Características

- Selección de aerolínea
- Cálculo de dimensiones en CM o Inches
- Cálculo automático de perímetro
- Generación de bundles

## Demo en vivo

[Ver demo](https://tu-usuario.github.io/airline-bundle/)

## Desarrollo local

```bash
python3 -m http.server 3000
```

Luego abre `http://localhost:3000`
